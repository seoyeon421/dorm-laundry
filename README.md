# 🧺 백석대학교 기숙사 세탁기 예약 시스템

## 📌 프로젝트 소개
기숙사 세탁기 사용 불편을 해결하기 위해 만든 예약 시스템입니다.

## 💡 주요 기능
- 세탁기 1, 2: 날짜 + 시간 예약 기능 (최대 7일)
- 세탁기 3, 4: 실시간 사용 상태 확인
- 로그인 기능 (학번 / 비밀번호 / 방번호)
- 마이페이지 (예약 상태, 패널티, 이용 내역)
- 패널티 시스템 (노쇼 제한)

## 🛠 사용 기술
- Python
- CustomTkinter (GUI)
- Pillow (이미지 처리)

## 🚀 실행 방법
```bash
python -m pip install customtkinter pillow
python main.py

## 🎯 프로젝트 목표
기숙사 내 세탁기 사용의 비효율성과 노쇼 문제를 해결하기 위해  
날짜 및 시간 기반 예약 시스템과 패널티 기능을 구현하여  
사용자 편의성과 운영 효율성을 향상시키는 것을 목표로 합니다.
